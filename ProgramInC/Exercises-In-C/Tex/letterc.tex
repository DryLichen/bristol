\nsection{Letter C}

Execute this program so you understand the output:

\begin{codesnippet}
#include <stdio.h>

#define HEIGHT 17

int main(void)
{

   int i = 0;

   printf("\n\nIIIIIII\n");
   while(i < HEIGHT){
      printf("  III\n");
      i = i + 1;
   }
   printf("IIIIIII\n\n\n");
   return 0;

}
\end{codesnippet}

\begin{exercise}
Write a similar program
that prints a large letter C on the screen (it doesn't need to be curved!).
\end{exercise}
